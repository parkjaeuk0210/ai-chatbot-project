# AI Chatbot Project

## 로컬 실행 방법 (Vercel과 동일한 환경)

1. 의존성 설치:
```bash
npm install
```

2. 로컬 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 접속:
```
http://localhost:3000
```

## 특징
- 로컬 환경이 Vercel 배포 환경과 동일하게 작동
- Express 서버로 정적 파일 제공
- ES6 모듈 지원
- API 엔드포인트 모킹 가능
